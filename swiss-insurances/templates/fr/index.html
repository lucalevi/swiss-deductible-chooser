Économisez sur votre assurance maladie en Suisse

Choisissez la bonne franchise. Économisez des centaines de francs par an. Insurek vous montre en quelques minutes quelle franchise est la plus avantageuse pour vous.

Comment ça marche – 3 étapes simples

  1. Obtenez vos primes : via le calculateur officiel du Conseil fédéral.
  2. Connaissez vos frais de santé annuels : moyenne des 3 dernières années.
  3. Entrez les données : Insurek vous indique la franchise qui vous fait économiser le plus.

Étape 1 – Obtenez vos primes (Priminfo)

Utilisez le calculateur officiel des primes (Priminfo) pour trouver les six primes mensuelles les moins chères dans votre région.

Sur Priminfo :

  1. Entrez votre code postal ou votre commune.
  2. Ajoutez votre année de naissance.
  3. Choisissez une franchise (n’importe laquelle pour commencer).
  4. Indiquez si vous avez besoin d’une assurance accidents (souvent non si vous travaillez +8 h/semaine).
  5. En bas de page, choisissez les modèles d’assurance (Standard, HMO, etc.).

Cliquez sur « Calculer ». Ensuite, ouvrez l’onglet « Comparaison des franchises » et cliquez sur 300 pour classer par primes les plus basses. Gardez la page ouverte pour la suite.

Étape 2 – Vos frais de santé moyens

Rassemblez vos factures médicales des 3 dernières années (consultations, médicaments, hospitalisations – sans les primes d’assurance). Additionnez le tout et divisez par 3. Cette moyenne vous aide à choisir la franchise la plus économique.
